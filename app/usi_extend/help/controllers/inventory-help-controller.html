Detail information

Inventory

User can get SSD, Cable, PowerSupply, USP, localhost IP, Fan, Switch and peer BMC information when mouse move on the corresponding location of pictures.


Overview

Click overview menu, the page displays network, time information, BMC firmware version , PCIe switch firmware version information and lastest 5 high logs. Click edit button, user can edit hostname. Click the LED switch can control LED on or off. click the serial over LAN console link, event log link, edit network settings link or help link, it can jump to responding page.


Sensors

The page displays all sensors information. Click export button can download the sensors data named sensors.json. Privide filter function, user can though key value to serach sensors information.


Hardware status

The page displays all hardware information. Click export button can download the hardware data named hardware_inventory.json. Privide filter function, user can though key value to serach hardware information.


Partition topology

The page displays partition information.


Event log

The page displays all event logs information. User can select one or more logs to copy, delete, mark as resolve and export. Privide filter function by key value, severity, data range and event status, user can though these functions to search logs.


System log

The page displays all system logs information. User can select one or more logs to copy, delete, mark as resolve and export. Privide filter function by key value, severity, data range and event status, user can though these functions to search logs.
