Sensors


Monitor all sensors status. If the sensor is abnormal, the current value background in the dashboard will become red, and if it is normal status, the current value background is green. Click export button will download sensors data in json format. Provide two methods to search sensors: 1) filter by key word: input key word user wants to search and click filter button, it will list search result. 2) filter by severity: choose All, Critical, Warning or Normal button to search. Hardware status


Click export button will download hardware status data in json format. Input key value, user wants to search and click filter button will list search result. Click the arrow button will show or hide the detail information.


Partition topology

The page displays partition information.


Event log

Monitor and display event logs. User can choose time zone to get time. It provides four functions to filter logs. 1) filter by key word: input key word and click filter button will list result. 2) filter by date: input start time and end time will list logs generated during the date. 3) filter by severity: choose All, High, Medium or Low button to search. 4) filter by event status: click the event drop-down menu to select.



System log

Monitor and display system logs. Click select system log type menu to choose system log type. It provides four functions to filter logs. 1) filter by key word: input key word and click filter button will list result. 2) filter by date: input start time and end time will list logs generated during the date. 3) filter by severity: choose All, Critical, Warning or Ok button to search. 4) filter by type status: click the type drop-down menu to select.
