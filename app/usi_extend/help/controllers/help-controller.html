Help information

Inventory

Your can get SSD, Cable, PowerSupply, USP, localhost IP, Fan, Switch and peer BMC information when mouse move on the corresponding location.


Switch power control

When power off switch, SSD will not work and PCIe switch firmware will not update.


PCIe switch firmware manage

When switch firmware Upload, first click update button to update image. This operation will take two minutes. Then, click activate button to activate image to work.


BMC firmware manage

When BMC firmware Upload, click activate button to activate image. This operation will take a few minutes.


Fan control

Only root user can set fan speed. When perform this operation the fan will rotate according to the set value instead of the BMC temperature.


Account manage

Only root user can create, edit and delete account.
