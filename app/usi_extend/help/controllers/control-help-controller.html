Detail information

LED control

Click Control LED control menu, user can control LED on or off though switch button.


Fan control

Only root user can set fan speed. User can set one fan speed also can set all fan speed one time. When perform this operation the fan will rotate according to the set value instead of the BMC temperature.


System control

When reboot AFA, Web browser loses contact with the AFA for several minutes. When the AFA is back online, you must log in again. When SSD and switch firmware update work, make sure switch is power on.


Serial over LAN console

Click Control Serial over LAN console menu,


BMC firmware

User can upload BMC image though location also can download image though TFTP server. When BMC image upload, user can delete image and also can activate it. When click activate button to activate image. This operation will take a few minutes.


PCIe switch firmware

User can upload PCIe switch image though location. When PCIe switch image upload, user can delete it and can update it. When click update button, it will take a few minutes. Update finish then can activate the image, it will also take a few minutes.


Network setting

User can modify hostname, network interface, MAC address and default gateway. AFA use DHCP IP defaultly, but user can set static IP though choosing radio button. Carefully, when set static IP, gateway is consistent with default gateway.


Date and time setting

User can set AFA time, host time separately. Also can set both or split at one time.


Account manage

Only root user can create, edit and delete account. Failed lockin times is unlimited and lockout times is none defaultly. Root user can set the failed lockin times between 3 to 10, and failed lockin times between 30s to 600s.
